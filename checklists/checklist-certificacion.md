# Checklist de certificacion

## Accesos

- [ ] Usuario administrador tiene acceso al portal SII.
- [ ] Firmante tiene certificado digital vigente.
- [ ] Firmante esta autorizado para firmar documentos.
- [ ] Firmante esta autorizado para enviar documentos.
- [ ] API key SimpleAPI disponible localmente.

## Folios de certificacion

- [ ] CAF 33 descargado.
- [ ] CAF 34 descargado, si aplica.
- [ ] CAF 46 descargado, si aplica.
- [ ] CAF 56 descargado.
- [ ] CAF 61 descargado.
- [ ] CAF guardados fuera de Git.

## Set de pruebas

- [ ] Set descargado desde SII.
- [ ] DTE generados con folios correctos.
- [ ] DTE firmados.
- [ ] Sobre EnvioDTE generado.
- [ ] Sobre enviado.
- [ ] TrackID consultado.
- [ ] Portal muestra conformidad.

## Libros

- [ ] Libro ventas generado.
- [ ] Libro compras generado.
- [ ] Ambos aceptados o marcados conformes en portal.

## Simulacion

- [ ] DTE simulacion generados.
- [ ] Sobre enviado.
- [ ] TrackID conforme.
- [ ] Portal permite avanzar.

## Intercambio

- [ ] XML de intercambio descargado.
- [ ] Respuesta de intercambio generada.
- [ ] Recibo de mercaderias/servicios generado.
- [ ] Resultado comercial generado.
- [ ] Archivos subidos con tic verde.

## Muestras impresas

- [ ] Lista del portal revisada.
- [ ] PDFs tributarios generados.
- [ ] PDFs cedibles generados cuando el portal los pide.
- [ ] Todos los PDFs tienen una pagina.
- [ ] Cedibles contienen texto `CEDIBLE`.
- [ ] No hay PDFs antiguos en la carpeta final.
- [ ] Todas las filas tienen tic verde.
- [ ] Boton `Enviar al SII` habilitado.
- [ ] Muestras enviadas a revision.

## Cierre

- [ ] Revision SII conforme.
- [ ] Declaracion de cumplimiento realizada.
- [ ] Autorizacion productiva recibida.
- [ ] CAF de produccion solicitados.
- [ ] Sistema cambiado a ambiente produccion.
