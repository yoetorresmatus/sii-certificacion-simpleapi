# Seguridad

Este repo esta pensado para ser publico.

## No publiques

- `.env`
- Certificados `.pfx` o `.p12`
- CAF XML reales
- XML DTE reales
- PDFs reales
- TrackIDs con RUT de empresa
- Claves SII, SimpleAPI u otros proveedores

## Reportar problemas

Si encuentras un secreto en el repositorio, abre un issue indicando el archivo y linea, sin copiar el secreto completo.

Si el secreto es tuyo, revocalo de inmediato.
